## How to run it standalone + inspect (no Docker needed)

```bash
mvn -pl collector -am package -DskipTests                 # build launcher
rm -rf /tmp/cryptopanner-real && mkdir -p /tmp/cryptopanner-real/segments
JAVA_OPTS="-Dconfig=$PWD/config/dev/real-binance.yaml" \
  collector/target/install/bin/collector | tee /tmp/cryptopanner-real/collector.log
# runs collector_max_runtime_s (config, currently 120s) then exits; or Ctrl-C (durable seal).
```
Data: `/tmp/cryptopanner-real/segments/<symbol>/<stream>/<date>/minute-<HH-MM>.jsonl.zst` (+`.sha256`).
Inspect: `zstd -dc <file> | head -1 | python3 -m json.tool`; verify `shasum -a 256 -c <file>.sha256`;
live metrics `curl localhost:8088/metrics`. Note: a minute file only appears after it seals
(boundary + 10s grace) or at shutdown.
